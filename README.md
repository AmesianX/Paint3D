# Paint3D

Remake of Microsoft Paint 3D

# Deploy

1. path-to-at-bin/windeploy.exe path-to-exeutable

2. copy the following folders from Qt installation directory:
    - Qt/labs/folderlistmodel
    - Qt/labs/settings
    - Qt/labs/sharedimage
    - QtGraphicalEffects
    - QtQuick/Controls
    - QtQuick/Dialogs
